How to find CPU Cores, VRAM & RAM?

CPU Cores: On Windows, open Task Manager → Performance → CPU. On Linux, lscpu. If unsure, count your logical cores.

GPU VRAM: Use the dropdown below (we have some known GPUs). Or check dxdiag (Win) / nvidia-smi (Linux).

System RAM: On Windows, right-click This PC → Properties. On Linux, free -h. On macOS, About this Mac.
